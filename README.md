AI MECHANIC FOR BIKES

A Full-Stack AI Application that bridges the gap between Technical Diagnostics and Financial Strategy.

ðŸ“– Overview

The AI Service Advisor is a smart kiosk designed for modern motorcycle repair shops. It solves a critical industry problem: The Trust Gap between mechanics and customers.

Instead of relying on guesswork, this system uses a Dual-Engine AI Architecture:

The Mechanic Brain: Diagnoses vehicle health using real-time sensor data (Voltage, Temp, Vibration).

The Banker Brain: Analyzes the customer's financial profile to check repair affordability.

The result is a data-driven Strategic Health Report that tells the customer not just what is broken, but how they should pay for it (e.g., Cash, EMI, or "Do Not Repair").

ðŸ“Š How It Works (The Logic)

The system uses a Dual-Brain approach:

1. The Mechanic Brain (Random Forest)

Input: Sensor readings (Air Temp, Process Temp, RPM, Torque, Tool Wear).

Training Data: Trained on the Kaggle Predictive Maintenance Dataset (10,000+ samples).

Logic: Detects non-linear failure patterns (e.g., High Vibration + Loose Chain = Drive Failure).

2. The Banker Brain (Linear Regression)

Input: Customer Work Experience (Years).

Training Data: Trained on Salary_Data.csv.

Logic: Predicts monthly income to calculate the "Financial Impact Score" (Repair Cost / Income).
